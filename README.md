## Getting Started

### Prerequisites

-   Install [Go](https://golang.org/doc/install)
-   Clone the repository:
    ```sh
    git clone https://github.com/vknow360/GoProjects.git
    cd GoProjects
    ```

## Usage

Each project is located in its own directory. Navigate to a project's folder and run it using:

```sh
cd project-name
go build project-name.go
```
